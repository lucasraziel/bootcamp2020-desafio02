### Installation Guide

Use this command to install:

```
git clone git@github.com:lucasraziel/bootcamp2020-desafio02.git desafio02

cd desafio02

npm install

```

You can test It by doing this command:

```
npm run-script test
```

You can start It by doing this command:

```
npm run-script dev
```
